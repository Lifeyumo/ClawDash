自动安装 OpenClaw

🚀 快速开始

一键安装

curl -s https://raw.githubusercontent.com/Lifeyumo/ClawDash/main/install.sh | bash

安装完成后访问：http://你的IP:99

手动安装

git clone https://github.com/Lifeyumo/ClawDash.git
cd ClawDash
npm install
node src/index.js

📋 命令

curl -s ... | bash 一键安装
curl -s .../uninstall.sh | bash 一键卸载
curl -X POST http://localhost:99/api/status 查看状态

🐳 Docker 部署

docker run -d -p 99:99 \
  -v ~/.openclaw:/root/.openclaw \
  --name clawdash \
  lifeyumo/clawdash

* 需要 OpenClaw 先在宿主机安装
